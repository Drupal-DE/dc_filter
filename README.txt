Simple module to provide some basic filters for http://drupalcenter.de.

----------
DC drupal.org project filter

This filter converts tags like [do:views] or [do:views Views Module] to links.
e.g. [do:views] would result in <a href="http://drupal.org/project/views" rel="external" title="drupal.org: views">views</a> 
and [do:views Views Module] would result in <a href="http://drupal.org/project/views" rel="external" title="drupal.org: Views Module">Views Module</a>
